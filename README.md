# Guru

AI 기반 맞춤형 교육 플랫폼
 LLM(LangChain)을 활용하여 사용자의 행동 데이터를 분석하고 맞춤형 강의를 추천하는 서비스입니다.

**[https://elioground.com/guru/](https://elioground.com/guru/)**

---

## 📚 프로젝트 개요

- **프론트엔드**: React, TypeScript, TanStack Query
- **백엔드**: NestJS, PostgreSQL
- **AI**: Python, LangGraph, FastAPI
